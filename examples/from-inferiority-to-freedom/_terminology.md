# 術語對照表（zh-TW）

本書統一用語，供撰寫與審稿參照。德文原文依阿德勒（Alfred Adler, 1870–1937）著作慣用詞彙保留。

| 原文（德文 / 英文） | 統一譯法 | 備註 |
|------|---------|------|
| Individualpsychologie / Individual Psychology | 個體心理學 | 阿德勒學派的正式名稱。不譯「個人心理學」避免與 personal 混淆。 |
| Minderwertigkeitsgefühl / Inferiority Feeling | 自卑感 | 普遍性、健康的經驗。不譯「劣等感」「自卑心態」。 |
| Minderwertigkeitskomplex / Inferiority Complex | 自卑情結 | 病理化的自卑感。與「自卑感」嚴格區分。 |
| Überlegenheitskomplex / Superiority Complex | 優越情結 | 自卑情結的補償過頭產物。 |
| Kompensation / Compensation | 補償 / 補償作用 | 不譯「彌補」。 |
| Organminderwertigkeit / Organ Inferiority | 器官自卑 | 身體限制帶來的早期自卑來源。 |
| Lebensstil / Style of Life | 生活風格 | 不譯「生活型態」以避免與 lifestyle 混淆。 |
| Frühe Kindheitserinnerungen / Early Recollections | 早期記憶 | 阿德勒臨床的「人格指紋」。 |
| Familienkonstellation / Family Constellation | 家庭星座 | 手足順位與家庭關係的結構。 |
| Gemeinschaftsgefühl / Social Interest / Community Feeling | 社會興趣 | 阿德勒學派最核心概念。常被誤譯為「社會情懷」「共同感」，本書一律用「社會興趣」以對應學界主流中譯。 |
| Lebensaufgaben / Life Tasks | 人生任務 | 三大人生任務：工作（Arbeit）、友誼（Freundschaft）、愛（Liebe）。 |
| Fiktionaler Finalismus / Fictional Finalism | 虛構目的論 | 亦譯「假想目的論」，本書採「虛構目的論」。 |
| Ermutigung / Encouragement | 鼓勵 | 與「讚美 / 誇獎（Lob）」嚴格區分。 |
| Entmutigung / Discouragement | 氣餒 | 阿德勒將神經症視為氣餒的狀態。 |
| Streben nach Überlegenheit / Striving for Superiority | 追求優越 / 向上奮鬥 | 注意：阿德勒後期將此解讀為「向上」而非「壓過別人」。 |
| Neurose / Neurosis | 神經症 | 不譯「神經質」。 |
| Zusammenarbeit / Cooperation | 合作 | 阿德勒人生觀的核心動詞。 |
| Mut / Courage | 勇氣 | 在阿德勒學派中，勇氣是「願意面對人生任務」的能力。 |

## 敘事規則

- 一律採旁白敘事：「阿德勒指出⋯⋯」「書中主張⋯⋯」「本書認為⋯⋯」。
- 引用阿德勒主張時，稱「阿德勒」「阿德勒醫師」或「本書作者」。
- 不以第一人稱「我」陳述阿德勒觀點。
- 避免使用岸見一郎《被討厭的勇氣》中的對白句式（「是的，沒錯」「青年說⋯」）以免讀者混淆。

## 常見誤用提醒

- **自卑感 ≠ 自卑情結**。前者是普遍經驗、推動力；後者是病理狀態。本書所有章節都必須維持此區分。
- **社會興趣 ≠ 合群**。前者是「願意為他人貢獻」的深層態度，後者是表面行為。
- **鼓勵 ≠ 讚美**。鼓勵指向努力與合作，讚美指向結果與比較。

## 引用格式

- 原著引用：blockquote 格式，附 `—— Alfred Adler, <Book Title>, <Year>, Ch./p.`
- 引用時單次 blockquote ≤ 200 字（依 config.yaml 的 `max_quote_chars`）。
- 引用原文若為英文譯本，註明 "translated by <translator>" 若已知。
